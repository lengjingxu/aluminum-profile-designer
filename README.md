# 🏗️ 铝型材结构设计器

> 铝型材结构设计与材料计算工具 - 微信小程序，支持3D可视化与配件清单生成

## 📖 项目简介

本项目是一款基于微信小程序的铝型材结构设计工具，面向定制家具（衣柜、书桌、展示架等）的设计师和DIY爱好者。用户可以通过2D绘图+3D实时预览的方式，快速设计铝型材框架结构，并自动生成材料清单与配件成本估算。

## ✨ 核心特性

- 📐 2D正交视图绘制 + 3D透视预览
- ⚙️ 内置铝型材规格库（2020/3030/4040等）
- 📊 自动计算型材长度与配件清单
- 💰 实时成本估算
- 🧩 预设模板与自定义组件库
- 📤 JSON/CSV/PNG 多格式导出
